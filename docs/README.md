# Documentation imSid Player

## Configuration Obsidian pour une meilleure lecture

### Option 1 : Paramètres Obsidian (Recommandé)

1. Ouvrez les paramètres d'Obsidian (`Ctrl+,` ou `Cmd+,`)
2. Allez dans **Apparence** → **Largeur maximale du contenu**
3. Désactivez la limitation ou augmentez la largeur maximale

### Option 2 : Mode Lecture

- Appuyez sur `Ctrl+E` (ou `Cmd+E` sur Mac) pour basculer en mode lecture
- Le mode lecture affiche généralement le contenu en pleine largeur

### Option 3 : CSS personnalisé

Si vous utilisez Obsidian, le fichier `.obsidian-styles.css` dans ce dossier contient des styles pour améliorer l'affichage des documents avec la classe `wide-content`.

Pour l'activer :
1. Créez un dossier `.obsidian` dans votre vault Obsidian (si ce n'est pas déjà fait)
2. Créez un dossier `snippets` dans `.obsidian`
3. Copiez le contenu de `.obsidian-styles.css` dans un nouveau fichier dans `snippets/` (par exemple `wide-content.css`)
4. Activez le snippet dans `Paramètres → Apparence → Snippets CSS`

Tous les diagrammes utilisent le format **Mermaid**, compatible avec :
- Obsidian (natif)
- GitHub (natif)
- VS Code (avec extension)
- La plupart des lecteurs Markdown modernes
